# Finals Submission Guide

## Full Application

This folder already includes a working full application with database connection and CRUD operations for all entities in the employee management ERD.

Run it with:

```powershell
npm run dev
```

Then open:

```text
http://localhost:3000
```

## Program Requirement

- Web-based application
- Connected to SQL database using Supabase Postgres
- Uses relational tables based on the employee ERD
- Supports add, edit, delete, and view operations
- Includes SQL query outputs for reporting

## Functional Requirement

- Manage department records
- Manage position records
- Manage employee records
- Manage attendance records
- Manage leave type records
- Manage leave request transactions
- Manage project records
- Manage employee project assignments
- View dashboard summaries
- Run and display SQL reports
